# Seshadri S Portfolio

This project contains the source code for a modern, responsive portfolio website built with React and Three.js (React Three Fiber). It showcases embedded systems and IoT work with interactive 3D visuals.

## Setup

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

## GitHub Codespaces

This project includes a `.devcontainer` folder so you can open it directly in
GitHub Codespaces. When the Codespace launches, dependencies will install
automatically and you can start the development server with:

```bash
npm run dev
```

## Customization

Replace placeholder images and add your own project information in the files under `src/components`. The 3D model in the hero section can be customized inside `Hero.jsx`.
